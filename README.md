# 🕹️ 16-Bit Retro Arcade Portfolio

A production-grade, highly interactive personal portfolio built with a **Retro Arcade Cabinet** aesthetic. This project features dual-mode functionality, allowing visitors to switch between an immersive **Arcade Experience** and a clean **Professional Dashboard**.

## ✨ Features

- **Dual-View System**: 
  - **Arcade Mode**: Full pixel-art immersion with RPG-style dialogue, character stats, and warp-pipe project navigation.
  - **Dashboard Mode**: A minimalist, high-contrast professional resume view for quick reading.
- **Strict Landing Gate**: A custom terminal barrier that requires users to select their preferred experience upon entry.
- **Dynamic Mode Persistence**: Seamlessly switch between modes while maintaining your scroll position and context.
- **Retro Physics**: Parallax cloud layers, scanline overlays, and pixel-perfect borders built with pure CSS.
- **Responsive Design**: Fully optimized for both desktop and mobile retro enthusiasts.

## 🛠️ Tech Stack

- **Frontend**: React 18 + TypeScript
- **Styling**: Tailwind CSS (Custom Retro Design System)
- **Animations**: Framer Motion
- **Bundler**: Vite
- **Fonts**: Press Start 2P (Pixel Font) & Monospace System Fonts

## 📦 Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/anassheikh12/Portfolio.git
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

4. Build for production:
   ```bash
   npm run build
   ```
