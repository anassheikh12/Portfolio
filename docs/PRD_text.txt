PRODUCT REQUIREMENT DOCUMENT (PRD)PROJECT: 16-BIT RETRO ARCADE PORTFOLIO (2026)SYSTEM OVERVIEWA highly structured, fast-loading personal developer portfolio framed as a classic 16-bit arcade cabinet. Taking aesthetic cues from nostalgic gaming titles (Sonic, Mario, Minecraft), the site delivers visual impact through clean flat pixel art, floating background clouds, and snappy CSS/Framer Motion animations without sacrificing professional readability or performance. The site operates in dual modes (Arcade vs. Clean Dashboard) to cater to different visitors.1. OBJECTIVES & TARGET AUDIENCEThe primary goal of this portfolio is to establish Anas Sheikh's technical capability as an incoming sophomore student at FAST-NUCES Karachi. It showcases functional, low-level coding proofs (C++, SFML, WebSockets, Web3) behind an engaging retro interactive interface designed for hiring managers and technical recruiters.2. GLOBAL STYLING & COLOR SCHEME• Sky Layer Background: Solid retro sky blue (#5C94FC).• Ground & Frame Borders: Textured pixel dirt brown (#865126) with solid black outlines (#000000).• Grass / Active Elements: Solid grass green (#00A800) with deep green shadows (#005C00).• Cloud Elements: Blocky, solid white (#FFFFFF) panels drifting infinitely in the background.• Outlines & Shadows: High-contrast thick borders (border-4 or border-8) with solid, hard offset shadows.3. SECTION-BY-SECTION FLOW• Fixed HUD Navigation Bar: Pinned header with navigation links ([ABOUT], [SKILLS], [PROJECTS], [CONTACT]) and dynamic sliding active cursor triangle (▶).• Screen 1: Choice Terminal: Center menu panel toggle allowing the user to select either 'Cooperative Arcade Mode' (keyboard interactive) or 'System Dashboard Mode' (minimalist dark developer resume).• Screen 2: Character Select (About Me): Grid display linking a pixel developer avatar card alongside an RPG dialogue bubble containing your bio.• Screen 3: Inventory Grid (Skills Menu): A blocky 4x2 grid of slots representing tool sets (C++, OOP, React, etc.). Selecting a slot updates the lower active stats panel describing technical metrics and projects.• Screen 4: Projects (Warp Pipe Select): Projects displayed as signature green warp pipes on a grassy horizon, triggering detailed gameplay loops and repository links inside descriptive overlay modals.• Screen 5: Contact Me (High Score Board): A scoreboard input block to capture recruiter name, email, and message with an output 'STAGE CLEAR' success screen.
